Realm

Realm is a cross-platform, open-source database replacement for SQLite on the Android operating system. Realm is easy to use and supports the same high-level models across Java and Swift&Objective-C. This makes it easy for developers in companies to share code between mobile teams.

Realm uses a zero-copy design so it performs faster than raw SQLite. We have an example below, to show you how easy it is to use Realm to store database information and access it quickly.
