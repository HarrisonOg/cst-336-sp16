On Android Development

Android natively uses SQLite to handle database transactions. SQLite and frameworks to use SQLite tend to be a bit messy to use for people not totally familiar with database management. Because of this, there are many developers who prefer to use other database solutions for storing data on the device. One of these solutions is Realm. Realm was developed by Square and released to open source.

Developers have many different solutions for doing HTTP API requests. Usually they are handled with messy asynchronous threads and functions. Retrofit is a handy solution for handling API requests and allows you to customize your own interfaces and classes for the android apps.

Both will be discussed in further pages.
